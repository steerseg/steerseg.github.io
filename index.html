SteerSeg

Attention Steering for Reasoning Video Segmentation

Ali Cheraghian · Hamidreza Dastmalchi · Abdelwahed Khamis · Morteza Saberi · Aijun An · Lars Petersson
📄 Paper soon 𝑥 arXiv soon 💻 Code soon 📊 Data soon
TL;DR

SteerSeg steers attention at its source by training a small set of learnable soft prompts on a frozen LVLM, paired with reasoning-guided Chain-of-Thought. This produces spatially concentrated attention maps that translate into accurate point prompts for reasoning video segmentation.

Highlights

🎯

Steer at the source

Reshape attention via input-level conditioning instead of post-hoc patching of attention maps.

🧠

CoT-guided reasoning

Chain-of-Thought attributes disambiguate similar objects and direct attention to the correct instance.

✨

Frozen LVLM, learnable prompts

Keeps the LVLM and segmentation model frozen; only a small set of soft prompts is trained, preserving pretrained reasoning while adapting grounding.

🚀

Strong generalization

Trained only on Ref-YouTube-VOS, yet generalizes across diverse video segmentation benchmarks.
